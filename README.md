# 基于auto.js开发的用于检测蚂蚁森林单删好友名单
# 使用须知：
## 1.为了便于控制程序，请打开auto.js设置允许使用音量上键强制停止程序
## 2.本脚本需要android 7.0以上系统使用
## 3.本脚本需要开启无障碍模式
